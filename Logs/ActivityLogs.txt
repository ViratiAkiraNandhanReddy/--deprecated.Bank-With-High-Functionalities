Activity Logs
  * Purpose: To track user actions and application events during the setup process.
  * File Name: ActivityLogs.txt
  * Content:
      * Timestamp of the activity.
      * Description of the action or event.
  * Example:
        [INFO]:[Setup.py][15-May-2025 @ 10:30:00 AM] - User selected SQLite3 as the database type.
        [INFO]:[Setup.py][15-May-2025 @ 10:35:00 AM] - Manager data submitted successfully.
