Error Logs
  * Purpose: To record any errors or exceptions that occur during the execution of the application.
  * File Name: ErrorLogs.txt
  * Content:
      * Timestamp of the error.
      * Error type and message.
      * File and function where the error occurred.
  * Example:
        [ERROR]:[Setup.py][15-May-2025 @ 10:45:23 AM] - Failed To Connect To MySQL Server; ErrorType: [mysql.connector.Error]
